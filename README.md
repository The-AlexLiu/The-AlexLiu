# Alex Liu

增长运营 / 广告投放 / 内容策略 / 数据分析

我的职业路线是：平台侧增长运营 → Agency 广告运营 → 品牌侧全域增长运营。

我主要关注如何把模糊的业务问题拆成可执行的增长实验、广告账户结构、内容生产流程、落地页诊断方法和数据复盘机制。

## 我关注的方向

| 方向 | 我主要解决的问题 |
|---|---|
| 广告投放 | Meta Ads、Google Ads、素材测试、账户结构、ROAS / CPA / GMV 复盘 |
| 内容增长 | 短视频素材结构、内容选题、达人素材、平台内容反馈和素材库沉淀 |
| 独立站转化 | 广告入口、落地页首屏、信任证据、CTA 和购买阻力诊断 |
| 数据分析 | 广告日报、素材表现、Feed 表格、UTM、GA4 和业务指标对齐 |
| AI 运营 | 用 AI 和自动化工具提高内容生产、数据整理和重复流程效率 |

## 作品集

这些 GitHub 仓库不是单纯用来放代码的项目。

我更希望它们像一组“认知仓库”：把真实运营工作中的判断方法、指标口径、检查清单、模板和可复用流程沉淀下来。

完整目录：
